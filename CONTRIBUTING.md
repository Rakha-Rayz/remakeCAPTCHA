# Contributing to remakeCAPTCHA

Thank you for your interest in contributing to this project.

We welcome improvements, bug fixes, and suggestions that help make this CAPTCHA system stronger and more secure against bots.

---

## üìå How to Contribute

1. Fork this repository.
2. Clone your fork to your local machine:
   ```
    git clone https://github.com/Rakha-Rayz/remakeCAPTCHA.git
    cd remakeCAPTCHA
    ```
3. Create a new branch:
    ```
   git checkout -b feature/your-feature-name
   ```
4. Make your changes.
5. Commit your changes with a clear message:
    ```
   git commit -m "Add: short description of changes"
   ```
6. Push to your fork:
    ```
   git push origin feature/your-feature-name
   ```
7. Open a Pull Request to the main repository.
